Capabilities

Everything you need,
nothing you don't

A complete terminal solution with powerful features designed for developers, sysadmins, and DevOps engineers.

Multi-Protocol

SSH, Telnet, Serial/COM, Local terminal. Connect to anything from a single interface, with full protocol-specific configuration.

Encrypted Vault

AES-256 encrypted credential storage. PIN, biometric, and FIDO2 security keys. Auto-lock. Your passwords never leave your machine.

Tabs & Split Panes

Multi-tab interface with vertical and horizontal splits. Resize freely and organize your workspace exactly how you want it.

SFTP Browser

Browse, upload, and download files with an integrated file manager. Drag & drop support and real-time file watching included.

Plugin System

Extensible via a manifest-based architecture. Add custom panels, commands, sidebar sections, context menus, and more.

Port Forwarding

Create SSH tunnels with local port forwarding. Monitor active tunnels and manage your forwarding rules with ease.

Protection

Security first

Your credentials are encrypted locally with AES-256. SimplyTerm supports multiple unlock methods and never sends your data anywhere.

  • Master password protection
  • PIN quick unlock (4-6 digits)
  • FIDO2 security keys (YubiKey, SoloKey, Titan)
  • Windows Hello / Touch ID biometrics
  • Auto-lock with configurable timeouts
  • Maximum security mode (re-lock after each connection)

Productivity

A terminal that
works for you

  • Command Palette Quick access to every action with Ctrl+Shift+P
  • Terminal Search Find anything in terminal output with regex support
  • Customizable Appearance Themes, fonts, cursor styles, window effects (Acrylic, Mica)
  • Keyboard Shortcuts Full keyboard-driven workflow for maximum efficiency
  • Session Folders Organize connections by project, environment, or team
  • Host Key Verification MITM protection with fingerprint verification on first connect
